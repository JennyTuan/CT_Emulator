# Maintenance & Calibration PRD

## 1. 功能概述

本模块负责 CT 设备的日常维护与校准，包含两个独立的核心流程：**球管预热 (Tube Warm-up)** 和 **空气校正 (Air Calibration)**。

## 2. 功能详细说明

### 2.1 球管预热 (Tube Warm-up)

* **目的**：当球管热容量过低时，通过预热保护球管并确保扫描质量。
* **输入项**：
  * `当前热容量`：实时显示当前球管的热容量状态（只读）。
  * `目标热容量`：用户可设定的预期预热目标百分比。
* **逻辑**：点击开始后，系统模拟增加热容量直至达到目标值。

### 2.2 空气校正 (Air Calibration)

* **目的**：校正探测器在不同物理参数下的响应一致性。
* **配置参数**：
  * `旋转速度`：支持 1s, 2s, 0.75s。
  * `焦点`：Small, Big。
  * `电压`：80kV, 100kV, 120kV, 140kV。
  * `准直器宽度`：32*0.6。
* **统计**：显示当前组合的总数、已完成数和待校正数。
* **操作**：支持清空校正记录和启动自动化校正流程。

## 3. 界面交互

* 采用 Tabs 标签页进行物理隔离，强调两者为不同的业务流程。
* 使用医疗级专业配色：蓝色 (#1890ff) 为主色调，预警色 (#faad14) 用于提示。
* 右侧配套实时系统日志，监控完整执行过程。
